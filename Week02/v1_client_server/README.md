# V1: Client-Server

## Nguyên tắc: Client-Server

**Đặc điểm:**
- Tách biệt Client và Server
- Server xử lý business logic và data
- Client xử lý UI và user interaction
- Giao tiếp qua HTTP

**Chạy:**
```bash
python app.py
```

**Test:**
- Mở http://localhost:5000
- Click "Load Books" để test client-server communication
